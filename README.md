# Light slider - простой, легковесный слайдер(такой нужен в основном если вы рендрите очень много слайдеров на одной странице)

## Project setup
```
npm install
```

### Webpack dev server
```
npm run serve
```

### Rullup build project
```
npm run rollbuild
```

### Lints and fixes files
```
npm run lint
```
